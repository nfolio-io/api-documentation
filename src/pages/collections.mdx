# Available Collections

This page provides information about the Collections endpoints, which allow users to see the available NFT Collections in NFolio {{ className: 'lead' }}

## The Collection model

The Collection model in NFolio includes the slug and asset contract address of the available collections in NFolio. This information can be used to filter collections and identify projects that can be evaluated using NFolio data.

### Properties

<Properties>
  <Property name="slug" type="string">
    Unique identifier of the collection.
  </Property>
  <Property name="token_id" type="string">
    The token id for the asset withing its collection.
  </Property>
</Properties>

---
