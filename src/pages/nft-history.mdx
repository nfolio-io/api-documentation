# NFT History

This page provides information about the NFT History endpoints, which allow you to access detailed data about your NFT assets programmatically. Use these endpoints to track and analyze the history of your assets {{ className: 'lead' }}

## The NFT History model

The NFT History model in napp.ai includes data on the historical estimated price of your NFTs at the intra-daily level. This information can be useful for tracking the value of your assets over time and making more informed investment decisions.

### Properties

<Properties>
  <Property name="asset_id" type="string">
    Unique identifier for the asset.
  </Property>
  <Property name="token_id" type="string">
    The token id for the asset withing its collection.
  </Property>
  <Property name="asset_contract_address" type="string">
    The contract address of the asset.
  </Property>
  <Property name="result" type="string">
    The result object with historical details on the estimated price of your
    asset.
  </Property>
</Properties>

---

## Get NFT Estimates History {{ tag: 'GET', label: '/v2/nft_history/{slug}/{token_id}' }}

<Row>
  <Col>

    This endpoint allows you to retrieve the historical estimated price of the specified NFT by slug and token id.

    ### Required attributes

    <Properties>
      <Property name="slug" type="string">
        Unique identifier of the collection.
      </Property>
      <Property name="token_id" type="string">
        The token id for the asset withing its collection.
      </Property>
    </Properties>

    ### Optional attributes

    <Properties>
      <Property name="from_date" type="string">
        The date used to filter the results from the specified date 'YYYY-MM-DD'.
      </Property>
      <Property name="to_date" type="string">
        The date used to filter the results to the specified date 'YYYY-MM-DD'.
      </Property>
      <Property name="limit" type="string">
        Limit the number of contacts returned.
      </Property>
      <Property name="offset" type="string">
        The desired number of results to skip.
      </Property>
    </Properties>

  </Col>
  <Col sticky>
